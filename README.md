# Music-Api

Node JS music Api

<hr>
Hashed paswword using: ```npm i bcrypt``` <br>
Jwt plugin: ```npm i @hapi/jwt``` <br>
get random string: access ```node``` -> input ```require('crypto').randomBytes(64).toString('hex');```

## before run do

npm install <br>
npm run migrate up

## run

npm run start-dev
